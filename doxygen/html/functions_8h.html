◆ FGETS()

char * FGETS ( char *  str,
int  num,
FILE *  stream 
)

Функция fgets считывает символы из файла и сохраняет их в виде строки в параметр str.

Parameters
[in]strУказатель на массив, в который сохраняются считанные символы.
[in]numМаксимальное количество символов для чтения, включая нулевой символ.
[in]streamУказатель, который идентифицирует поток, из которого считываются символы.
Returns
Указатель на строку назначения(NULL в случае ошибки)

Символ новой строки прекращает работу функции fgets, но он считается допустимым символом, и поэтому он копируется в строку str.

Нулевой символ автоматически добавляется в строку str после прочитанных символов, чтобы сигнализировать о конце строки.

◆ GETLINE()

size_t GETLINE ( char **  lineptr,
size_t *  num,
FILE *  stream 
)

Функция считывает целую строку, сохраняя адрес буфера, содержащего текст, в *lineptr.

Parameters
[in]strУказатель на копируемую строку.
[in]lineptrДвойной указатель на копируемую строку.
[in]streamУказатель на файловый поток.
Returns
Указатель на строку назначения(NULL в случае ошибки)

Буфер завершается null и содержит символ новой строки, если был найден разделитель для новой строки.

Если *lineptr равно NULL, то процедура будет создавать буфер для содержимого строки, который затем должен быть высвобожден программой пользователя. В любом случае при успешном вызове *lineptr и *num будут обновлены для отражения адреса буфера и его размера соответственно.

Функция добавляет первые num символов строки srcptr к концу строки destptr, плюс символ конца строки.

Parameters
[in]srcptУказатель на копируемую строку.
[in]destptrУказатель на изменяемую строку.
[in]numМаксимальное количество символов для конкатенации.
Returns
Указатель на строку назначения.

Если строка srcptr больше чем количество копируемых символов num, то после скопированных символов неявно добавляется символ конца строки.

◆ STRNCPY()

char * STRNCPY ( char *  destptr,
const char *  srcpt,
size_t  num 
)

Функция копирует первые num символов из строки в строку.

Parameters
[in]srcptУказатель на копируемую строку.
[in]destptrУказатель на изменяемую строку.
Returns
Указатель на строку назначения.

Если конец строки srcptr (символ конца строки) достигнут прежде, чем были скопированы num символов, к скопированным символам в конец строки destptr добавляется нуль-символ, после чего строка считается скопированной. Если же строка назначения окажется меньше num, тогда скопируются символы, которые поместятся в destptr, учитывая то, что в конце строки обязательно должен стоять символ конца строки.
